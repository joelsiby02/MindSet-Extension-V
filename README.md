# 📚 Daily Learning System  
### Turn School Learning into Real Understanding

An AI-powered platform that helps students **retain what they learn in school** and gives parents **daily proof of understanding**.

---

## 🚀 What This Solves

Students forget what they learn within hours.  
Parents don’t know if their child actually understands anything.

**This fixes both.**

---

## ⚡ How It Works

1. Student enters what they learned (e.g., *Fractions*)
2. AI generates a simple, personalized lesson  
3. Student answers 5 thinking questions  
4. Takes a quick 3-question quiz  
5. Parent receives a **WhatsApp report with real answers + score**

## 🛠️ Installation

```bash
git clone https://github.com/yourusername/daily-learning.git
cd daily-learning
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
streamlit run main.py
```

